# Sydney #
Sydney is a fresh theme for Cotonti Siena.

## Installation ##
1. Place the folder 'sydney' in your Cotonti themes folder.
2. Enable the theme by setting: `$cfg['defaulttheme'] = 'sydney';` in datas/config.php.
3. Enjoy!
